# A JavaScript Template For Jasmine

This is a great way to start using Jasmine. This has been forked from the [Jasmine project](http://jasmine.github.io)

## Installation

Simply download and copy into any folder and repalce your src/*.js files and update the helpers/*.js with your changes.

Then open VS Code and add the extension '[Jasmine Test Explorer](https://marketplace.visualstudio.com/items?itemName=hbenl.vscode-jasmine-test-adapter)'

Then click on the beaker and you will see the list of test.

Click the play button and your tests will be running.
